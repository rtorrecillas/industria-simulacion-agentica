# Industria - Simulacion Agentica

Presentacion interactiva sobre optimizacion agentica de una planta industrial mediante un gemelo digital 3D, simulacion DES y datos MES/APS/SCADA/PLC.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El despliegue a GitHub Pages se ejecuta con GitHub Actions al hacer push a `main`.
